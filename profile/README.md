## What We Build

AiProducting ships real products. From expense splitting apps to global intelligence dashboards, from Kubernetes infrastructure to Terraform modules — this is where ideas become production systems.

### 💰 End-User Applications

| Project | Description | Tech |
|---------|-------------|------|
| **[split4us](https://github.com/AiProducting/split4us)** | Expense sharing — group tracking, balance calculations, settlement management | TypeScript |
| **[split4us-mobile](https://github.com/AiProducting/split4us-mobile)** | Split4Us mobile companion app | TypeScript |
| **[worldmonitor](https://github.com/AiProducting/worldmonitor)** | Global intelligence dashboard — AI news aggregation, geopolitical monitoring | TypeScript |
| **[worldmonitor-finance](https://github.com/AiProducting/worldmonitor-finance)** | Finance-focused WorldMonitor — market data and financial intelligence | TypeScript |

### 🏗️ Infrastructure & Platform

| Project | Description | Tech |
|---------|-------------|------|
| **[terraform-infrastructure](https://github.com/AiProducting/terraform-infrastructure)** | Production-ready modular Terraform — modules, templates, CI/CD | HCL |
| **[cncf-kubernetes-dashboard](https://github.com/AiProducting/cncf-kubernetes-dashboard)** | Web dashboard for 36+ CNCF projects in Kubernetes | JavaScript |
| **[redhat-middleware-stack](https://github.com/AiProducting/redhat-middleware-stack)** | Docker-based middleware — Kafka, Camel, WildFly, monitoring | JavaScript |
